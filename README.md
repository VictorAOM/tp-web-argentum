# Argentum Online — Presentación TP

Página estática para presentar el TP de Taller de Programación I usando un video embebido de YouTube.

## Cambiar el video

En `index.html`, buscar:

```html
https://www.youtube.com/embed/hjM7UCIFi2M
```

## Uso local

Ejecutar:

```bash
python3 -m http.server 8000
```

Abrir:

```text
http://localhost:8000
```
